# Лабораторные работы 

### Инструкция

``` 

Создаем pull-request, в корневой директории нужно (в самом начале) создать папку со своей фамилией.

В ней создать папку с номером лабы (соответственно все исходники должны быть в ней).

В папке с лабой (где у вас исходики) нужен README.md, в котором будет написано задание 
(чтобы мне не лезть каждый раз в этот длинный список). Также укажите, какое доп. задание было сделано

Вообще в идеале лучше, я думаю, сделать так: вы сделали лабу, залили ее сюда, а затем залили лабу со сделанным 
доп. заданием рядом с первой (чтобы можно было посмотреть, что изменилось)

Будет неплохо если включите уже скомпилированный jar-файл.

Можете кидать всю папку проекта целиком (которая в IDE).

```
